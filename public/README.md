# Public Assets Folder

Place your logo files here:

## Required Logo Files

1. **Consultare Logo**: `consultare-logo.png` (or `.jpg`, `.jpeg`, `.svg`)
   - Place your actual Consultare logo image file here
   - The system will try to load: .png → .jpg → .jpeg → .svg
   - Recommended size: 180x40px or similar aspect ratio

2. **SAP Business One Logo**: `sap-logo.png` (or `.jpg`, `.jpeg`, `.svg`)
   - Place your actual SAP Business One logo image file here
   - The system will try to load: .png → .jpg → .jpeg → .svg
   - Should include the SAP square logo and "Business One" text
   - Recommended size: 200x40px or similar aspect ratio

## Supported Formats

- PNG (preferred - will be tried first)
- JPG/JPEG
- SVG

## Instructions

Simply place your Consultare logo image file as `consultare-logo.png` (or .jpg/.jpeg) in this folder and it will be automatically used.
